# Terminal Gambling Game

A simple, addictive gambling game that runs in your terminal. Roll two dice and win when they match!

## Requirements
- Python 3.x
- Linux or macOS (uses `clear` command)

## How to Play
1. Clone this repository or download `game.py`
2. Run: `python3 game.py`
3. Press ENTER to roll
4. Match both numbers to win!
5. Press Ctrl+C to quit

## Features
- 11% win rate - rare enough to feel rewarding
- Instant gameplay with no lag
- No internet required
- Runs entirely in terminal
